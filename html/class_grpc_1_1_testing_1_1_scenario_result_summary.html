@type float $qps Total number of operations per second over all clients. What is counted as 1 'operation' depends on the benchmark scenarios: For unary benchmarks, an operation is processing of a single unary RPC. For streaming benchmarks, an operation is processing of a single ping pong of request and response. @type float $qps_per_server_core QPS per server core. @type float $server_system_time The total server cpu load based on system time across all server processes, expressed as percentage of a single cpu core.
