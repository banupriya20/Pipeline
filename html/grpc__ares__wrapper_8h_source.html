81// Asynchronously resolve \a name (A/AAAA records only).
82// It uses \a default_port if a port isn't designated in \a name, otherwise it
83// uses the port in \a name. grpc_ares_init() must be called at least once
84// before this function. The returned grpc_ares_request object is owned by the
85// caller and it is safe to free after on_done is called back.
86
87// Note on synchronization: \a as on_done might be called from another thread
88//~immediately, access to the grpc_ares_request* return value must be
89// synchronized by the caller. TODO(apolcyn): we should remove this requirement
90// by changing this API to use two phase initialization - one API to create
91// the grpc_ares_request* and another to start the async work.
